# 2.2 — Medidas DAX

## Conversión

```DAX
Conversion_Lead_to_App_% = DIVIDE([Applications], [Leads_Total])
```

## YoY

```DAX
Leads_YoY_% = 
DIVIDE(
    [Leads_Count] - CALCULATE([Leads_Count], SAMEPERIODLASTYEAR(leads_clean[created_at])),
    CALCULATE([Leads_Count], SAMEPERIODLASTYEAR(leads_clean[created_at]))
)
```

## Tiempo primera llamada

```DAX
Avg_Time_to_First_Call_Hours = 
AVERAGEX(
    leads_clean,
    DATEDIFF(leads_clean[created_at], leads_clean[First_Call_Date], HOUR)
)
```

## Duración llamada

```DAX
Avg_Call_Duration_Min = 
DIVIDE(SUM(calls[duration_sec]), COUNT(calls[call_id])) / 60
```
